Sales Data Analysis using Superstore Dataset

Project Overview

This project analyzes retail sales data from the Superstore dataset to uncover business insights, revenue trends, product performance, and profitability patterns.

The analysis helps businesses make data-driven decisions by identifying top-performing products, profitable categories, and sales trends over time.

---

Objectives

- Analyze monthly revenue trends
- Identify top-selling products
- Perform profit analysis
- Visualize business performance
- Generate actionable insights

---

Dataset

Dataset: Superstore Sales Dataset

Source: Kaggle

---

Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

Analysis Performed

1. Data Cleaning
- Date conversion
- Data validation
- Handling inconsistencies

2. Monthly Revenue Analysis
- Sales trend visualization
- Revenue growth analysis

3. Product Performance Analysis
- Top 10 selling products
- Product-wise sales comparison

4. Profit Analysis
- Category-wise profit analysis
- Monthly profit trends

5. Data Visualization
- Line Charts
- Bar Charts
- Business Insight Visualizations

Key Insights

- Identified peak sales periods.
- Discovered top-performing products.
- Evaluated category-wise profitability.
- Analyzed revenue and profit trends.
